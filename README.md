# Honeybee-Clicker
A bee-themed clicker game. Playable [here](https://p1x3l-w0lf.github.io/Honeybee-Clicker/).

## About This Project
I made this game in javascript, html (obviously), and css.
I'm pretty proud of it honestly, because I learned how to put images in my GUI design and it looks so much more polished and more professional looking.

Anyway... 
For anyone that's been living under a rock and doesn't know what "clicker game" means, your goal is to rack up as many points as possible by clicking on the prominent object in the center of the screen (in this case the button with the bee on it). You can also spend your points to buy upgrades from the shop, like an auto click upgrade and a buff to click strength. 

## What I Got From This Project
In making this project I learned...

• how to use an image for the background of a page

• how to put an image on a clickable element

• how to scale a webpage for diff device sizes

• how to have 2 things side by side on a page (it sounds easy, but this actually gave me a lot of heck [insert sobbing emoji here] I'm a beginner)

• how to have code run repeatedly without user interaction (aka, putting the auto in autoclicker)

## Modding
There are no traditional, downloadable, videogame "mods" available. But.

If you want to change the game up a little bit you can...

• adjust structural webpage elements in index.html (the .html is very important here, don't confuse it with the next file!)

• change up the game logic and that kinda thing in index.js (again, dont confuse this with .html)

• or change style.css to get the aesthetic of the game to your liking
